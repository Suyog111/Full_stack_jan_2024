Request -> **Api.js**
    if request starts from `/api/user`->    **userRouter**
    if request starts from `/api/product` ->   **ProductRouter**
    if any request does belong to above two -> **app.use in api.js**

**userRouter**

    GET  /:userid-> getUserById
    POST / -> createUser
    DELETE /:userId -> deleteUserById
